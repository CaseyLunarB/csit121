A Lesson on Forms in HTML

What are forms?

Forms in HTML are those areas that you type in on a website. Whether it be to log in to an account or sign up search something up on google, place in your card information, or completing a test, forms are a great tool that companies and even small businesses. Welcome to Watch Mojo and today I will be discussing the Top Five form elements needed to make visually functioning forms. (Disclaimer, these forms will not take in information unless a third party coding language, such as Javascript or Python is used.)

What do Forms do?

Forms are the first step in the submition process. Once a form is clicked or typed in and submitted, it becomess the job of boolean languages to maken those forms mean something.

Here are the Top Five Elements for creating a Form in HTML:

1. The Form Element
This is the basic element for starting a form in html. It is the start of the form area.
2. The input Element
This tag allows the website user to type in the designated area. From letters to words to symbols, this tag is a key element for password and log in forms.
3. The checkbox Element
This tag creates a different type of form. It allows the user to select options presented by the coder.
4. The textarea Element
This tag grants the user a space to type on the screen.
5. The selectarea Element
This tag grants the user to click a drop down menu as select an option.

There are many more elements that make many more elaborate forms, but these are the basic elements that help you get started.

Attributes

Each element possesses attributes that change the way a form is formatted, and/or what specific function it needs to do.

Examples of Each Element's Attributes

Element Attribute Function
Form name Names the form
input type Allows a user a box to type in
checkbox required Gives a required symbol for the
user to know the following form is required.
textarea maxlength Sets the limit of how long a user can type
selectarea size Sets the # of choices presented in a drop-down
